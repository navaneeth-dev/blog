# blog.rizexor.com

## Requirements

- Joplin
- RubyInstaller+Devkit
- bundler
- jekyll
- [Joplin Blog Explorter](https://github.com/rxliuli/joplin-utils)
- Joplin outline plugin
- NodeJS

## Getting started

- Clone
- `bundle install`
- `bundle exec jekyll serve`
- `npm i -D`

## Generate blogs

- Delete all in layouts
- `npm run clean`
- `npm run gen`
