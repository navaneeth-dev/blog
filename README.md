# blog.rizexor.com

## Requirements

- Joplin
- RubyInstaller+Devkit
- bundler
- jekyll
- [https://github.com/rxliuli/joplin-utils](Joplin Blog Explorter)
- Joplin outline plugin

## Getting started

- Clone
- `bundle install`
- `bundle exec jekyll serve`

## Generate blogs

- Delete all in layouts
- `npm run clean`
- `npm run gen`
